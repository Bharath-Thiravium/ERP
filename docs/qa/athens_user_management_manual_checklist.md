# Athens User Management Manual QA Checklist

- Project Admin creates employee (basic details only) and downloads credential slip
- Employee logs in with temporary password and is forced to reset it
- After reset, employee is redirected to profile completion
- Employee completes required profile fields + uploads documents and submits
- Project Admin sees the submission only for their project + company scope
- Project Admin approves the user profile
- Employee can access only induction page until induction is completed
- After induction completion, employee gets full access to Athens modules
