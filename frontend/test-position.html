Position Element Utility Test

This page demonstrates the JavaScript utility for positioning elements in corners using XPath.

Usage Examples:

// Basic usage
positionElementInRightCorner();

// Custom position
positionElementInRightCorner(null, {
    position: 'bottom-right',
    offset: { bottom: 30, right: 30 }
});

// Custom XPath
positionElementInRightCorner('/html/body/div[1]/div/div[1]/main/div/div[2]/div[1]/div[2]/div/div[1]');

// Auto-retry mechanism
autoPositionElement(5, 1000); // 5 retries, 1 second delay

The red element above will be positioned based on the XPath: /html/body/div[1]/div/div[1]/main/div/div[2]/div[1]/div[2]/div/div[1]

Features:

  • ✅ Position in any corner (top-right, bottom-right, top-left, bottom-left)
  • ✅ Custom offset values
  • ✅ Automatic body padding to prevent content hiding
  • ✅ Preserve original styles option
  • ✅ Retry mechanism for dynamic content
  • ✅ Restore original position
